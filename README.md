# Honduras Social

Landing page de la Fundacion Honduras Social, dedicada a la educacion, emprendimiento, inclusion y cooperacion para el desarrollo de comunidades en Honduras.

## Tech Stack

- **Astro 5** — Static site generator
- **Tailwind CSS v4** — Utility-first CSS
- **TypeScript** — Strict mode

## Comandos

| Comando             | Accion                                  |
| :------------------ | :-------------------------------------- |
| `npm install`       | Instalar dependencias                   |
| `npm run dev`       | Servidor local en `localhost:4321`      |
| `npm run build`     | Build de produccion en `./dist/`        |
| `npm run preview`   | Preview del build antes de deploy       |

## Estructura

```
src/
  components/    — Componentes de la landing (Hero, Navbar, etc.)
  layouts/       — Layout principal
  pages/         — Rutas (index.astro)
  styles/        — CSS global (Tailwind)
  assets/        — Imagenes y SVGs
public/          — Assets estaticos
```
